Meteo Watcher V

   Quinta generación del monitor meteorológico Meteo Watcher, de utilidad en el observatorio para control de: Nubosidad, Temperatura ambiente,  Presión atmosférica, Humedad Relativa, Punto de Rocío, Luminosidad y Lluvia
     

Elbrus_sync

   Herramienta para utilización de Elbrus desde CCD Commander
     

VNC

   Tutorial de cómo instalar y configurar la herramienta de conexión remota basada en VNC TightVNC.
     

Conversión de MJD a HJD

   En ocasiones, al intentar complementar los datos de las curvas de luz de una variable usando datos obtenidos en VSX, éstos tiene las fechas en formato MJD (Modified Julian Date) y es necesario convertirlas a HJD.
     

Occult_Shot

   Programa diseñado para la realización de fotos de ocultaciones mediante el método de la deriva, es decir, parando los motores de la montura. Interactúa con las monturas a través  drivers ASCOM y MaximDL como interfaz para las cámaras.
     

Park_Scope

   Herramienta para aparcar una montura desde CMD u otras aplicaciones, como puede ser CCD Commander.
     

Sendmail

   Herramienta para envío de alertas por e-mail desde CCD Commander
     

Virtual Weather Station

   Herramienta para utilización de una estación meteorológica remota desde CCD Commander
     

Autoguide

   Herramienta para autoguiado desde CCD Commander usando el algoritmo de MaximDL
     

Hiberna

   Herramienta para hibernar y despertar monturas que no dispongan de esta funcionalidad
     

Perfect Slew

   Herramienta que permite desplazamientos muy exactos usando Elbrus
